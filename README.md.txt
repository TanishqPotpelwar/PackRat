A simple file compression and decompression tool using Huffman Coding or Huffman Tree Implementation.

## 📌 Features
- Lossless compression
- Huffman Tree implementation
- Encode & Decode support
- File-based input/output

## 🛠 Tech Stack
- C++
- STL (priority queue, map)

## 📂 Project Structure
src/ → source code  
data/ → input/output files  

## ▶️ How to Run

```bash
g++ src/main.cpp src/huffman.cpp -o huffman
./huffman
